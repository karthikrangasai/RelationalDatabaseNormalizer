# A Relational Model Normalizer which normalizes up to BCNF

## Input Procedure:
* Relation: R(A1, A2, ..... , An)
* Functional Dependencies: Ai,Aj, ... ,Ak->Ap,Aq, ... ,Ar;Ai,Aj, ... ,Ak->Ap,Aq, ... ,Ar;Ai,Aj, ... ,Ak->Ap,Aq, ... ,Ar (Semicolon separated values)
